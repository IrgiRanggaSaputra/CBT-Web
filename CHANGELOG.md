# CBT LPK - Changelog

## Version 1.0.0 (December 2025)

### ✨ Initial Release

#### Fitur Peserta
- ✅ Login system dengan autentikasi
- ✅ Dashboard dengan daftar tes tersedia
- ✅ Halaman petunjuk tes
- ✅ Interface tes online dengan timer
- ✅ Autosave jawaban real-time
- ✅ Auto-submit ketika waktu habis
- ✅ Navigasi soal dengan indikator
- ✅ View hasil tes dan status kelulusan
- ✅ Profile management

#### Fitur Admin
- ✅ Dashboard dengan statistik lengkap
- ✅ CRUD Peserta
- ✅ CRUD Kategori Soal
- ✅ CRUD Bank Soal
- ✅ Import peserta dari CSV/Excel
- ✅ CRUD Jadwal Tes
- ✅ Assign peserta ke jadwal tes
- ✅ Monitoring real-time peserta
- ✅ Laporan hasil tes
- ✅ Export laporan ke Excel
- ✅ Penilaian otomatis

#### Technical Features
- ✅ PHP Native (no framework)
- ✅ MySQL Database
- ✅ Bootstrap 5 UI
- ✅ jQuery & AJAX
- ✅ Responsive design
- ✅ Session management
- ✅ Password hashing (bcrypt)
- ✅ SQL injection prevention
- ✅ XSS protection

#### Security
- ✅ Password encryption
- ✅ Session-based authentication
- ✅ Input sanitization
- ✅ CSRF protection (basic)
- ✅ Prevent browser back during test
- ✅ Disable right-click during test
